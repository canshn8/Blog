Foundation




Paket Yöneticileri ile yükleyin

Foundation npm, Bower, Meteor ve Composer'da mevcuttur. Paket, tüm kaynak Sass ve JavaScript dosyalarının yanı sıra derlenmiş CSS ve JavaScript'i sıkıştırılmamış ve sıkıştırılmış tatlarda içerir.

npm install foundation-sites
yarn add foundation-sites



Ardından yeni bir Vakıf projesi oluşturmak için kullanın:

foundation new

"Siteler için Temel"i seçtikten sonra, Foundation CLI size hangi şablonu kullanmak istediğinizi soracaktır. Şunlar arasından seçim yapabilirsiniz:




Temel şablon


Yeni başlayanlar için önerilir Foundation'ı kullanmaya başlamak için temel bir şablon. O içerir:

  • Foundation for Sites önceden yapılandırılmış.
  • Sass derlemesi
  • SASS/SCSS dosyalarınızı CSS'ye dönüştürmek için bir araç.
  • Başlangıç ​​HTML dosyası
  • Temel Foundation bileşenini kullanmanıza yardımcı olacak temel bir dosya (yeni XY ızgarası dahil!)



ZURB şablonu


Deneyimli (veya meraklı) kullanıcılar için önerilir Foundation ve aşağıdakileri içeren bir oluşturma sürecini içeren daha gelişmiş bir proje:

  • Panini ile gidon HTML şablonları
  • Sass derleme ve önek
  • Web paketi ile birlikte JavaScript modülü
  • Dahili BrowserSync
  • CSS, Javascript ve Görüntü sıkıştırma ile üretim oluşturma



Foundation 5 kullanıcıları


Makinenizde zaten Foundation 5 CLI varsa, komut satırı ortamınızın nasıl yapılandırıldığına bağlı olarak komutlardan yalnızca birine erişebilirsiniz.
Foundation 5 CLI'yi kaldırmak için gem uninstall foundation. Foundation 6 CLI'yi test ettikten sonra, eski CLI'ye geri dönmek için onu kaldırmak istiyorsanız npm uninstall foundation-cli --global.




Windows kullanıcıları

node-gyp aracı tarafından gerekli olduğundan, düğüm ortamınızda python v2.7'nin mevcut olduğundan emin olun. Bunu başarmanın iki yolu var

Windows-build-araçlarını kurun (önerilir) ve python'u daha sonra şuradan erişilebilir hale getirin:npm config set python "%USERPROFILE%\.windows-build-tools\python27\python.exe"

Python'u kurun (önerilmez) ve sistem ortamı değişkenlerinize ekleyin İlk yol, python v2.7'yi henüz yüklemediyseniz, makinenizi düğüm ortamı dışında etkilemediği için önerilir. Python v2.7'yi zaten yüklediyseniz, elbette her iki yolu da atlayabilir ve hemen Foundationc-cli'yi kullanmaya başlayabilirsiniz.
